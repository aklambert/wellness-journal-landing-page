# Journal Features

The **Journal Features** screen shows each of the journal related features available. Navigate to this screen by clicking the Journal icon in the bottom navigation:

<img src={journalFeaturesImg} width={240} />

- Click **Journal** to access the Journal feature, where you can save, edit, and delete journal entries

- Click **Goals** to access the Goals feature, where you can save, edit, and delete goals
