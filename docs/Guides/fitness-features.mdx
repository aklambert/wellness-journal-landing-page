# Fitness Features

The **Fitness Features** screen shows each of the fitness related features available. Navigate to this screen by clicking the Fitness icon in the bottom navigation:

<img src={fitnessFeaturesImg} width={240} />

- Click **Exercises** to access the Custom Exercises feature, where you can save, edit, and delete custom exercises

- Click **Workouts** to access the Custom Workouts feature, where you can save, edit, and delete custom workouts

- Click **Workout Logs** to view saved workout logs
